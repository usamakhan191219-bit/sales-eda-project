\# Sales Data Exploratory Data Analysis (EDA)



\## Project Overview

Performed comprehensive EDA on 5,000+ sales records to uncover customer behavior, sales trends, and satisfaction patterns.



\## Key Findings

\- \*\*Total Revenue\*\*: $6,363,773.40

\- \*\*Best Selling State\*\*: Texas ($1.35M)

\- \*\*Top Product Category\*\*: Clothing ($1.36M)

\- \*\*Average Customer Satisfaction\*\*: 2.99/5 (needs improvement)

\- \*\*Gender Split\*\*: 50% Male / 50% Female



\## Visualizations Included

\- Sales by State (Bar Chart)

\- Sales by Gender (Pie Chart)

\- Sales by Age Group

\- Product Category Performance

\- Customer Satisfaction Distribution



\## Tools Used

\- Python (Pandas, NumPy, Matplotlib, Seaborn)

\- Jupyter Notebook



\## How to Run

1\. Clone this repository

2\. Run `generate\_data.py` to create the dataset

3\. Open `sales\_eda\_analysis.ipynb` in Jupyter

4\. Run all cells
